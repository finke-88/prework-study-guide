HTML

  • The head element contains information about the webpage.
  • The body element represents the visible content shown to the user.
  • The paragraph element contains text and always starts on a new line.
  • The anchor element is used to link to external hyperlinks.
  • "< link rel="stylesheet" href="./assets/css/style.css" />" is used to link the style.css sheet to the index
  • Use sections < section > to space out the body of your HTML file and make it easier to add css.

CSS

  • A margin indicates how much space we want around the outside of an element.
  • A padding indicates how much space we want around the content inside an element.
  • Color values can be either semantic like blue or grey, or added with a hex color code.HTML Color Code website.
  • You can assign CSS to specific parts of a code with class id's.
  • .lesson { } or .title { } are CLASS attributes used in css to change the style of anything with that CLASS. They are always .attribute-name
  • #email { } or #github { } are ID tag attributes uesed in css to change only what is tagged with the ID. they are always #attribute-name
  • CSS can also be used to select semantic HTML elements like head, section or (p) paragraph
  • A period before the { } brakets means this is a universal CSS statment that will affect the whole page until overridden by another more fine tuned CSS attribute such as class or ID.
  • The class or ID must be in the line with what you want it to affect, so in the line with head, paragraph or even with list items.

Git

  • git status: checks what branch we are currently on.
  • git checkout -b branch-name: creates and switches to a new branch.

JavaScript

  • A variable is a named container that allows us to store data in our code.
  • Control flow is the order in which a computer executes code in a script.
  • A function must be called in the code to execute.
  • An array will always start with 0.
